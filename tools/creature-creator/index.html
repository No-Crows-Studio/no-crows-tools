Creature Creator

Add limb images, position them, and connect them with joints. Press + Joint to add a joint. Right-click near a joint and drag to another joint to connect them. Set each joint's type to shape and color its marker. Select a joint to check off the limbs and other joints it connects to. Hide a limb to lock it out of the way while you work. Select a limb or joint and nudge it with the arrow keys (hold Shift for larger steps). Export joint data writes each joint's offset from its connected limb centers to JSON. Export full creature saves a zip of the images and layout that import full creature can reload.
